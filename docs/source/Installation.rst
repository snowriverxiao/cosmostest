Installation
------------
.. code-block:: python

   git clone https://github.com/Lin-Xu-lab/COSMOS.git
   cd COSMOS

   If you would like to import COSMOS in different directories, there is an option 
   to make it work: 
   python setup.py install --user &> log


.. code-block:: python

   import Cosmos
